# Programação em Ambiente Web — PAW Food Hub

Aplicação web desenvolvida no âmbito da unidade curricular de **Programação em Ambiente Web**, com foco no desenvolvimento de uma plataforma completa de gestão de restaurantes.

## Objetivo
Criar uma aplicação web que permita:
- Registo e autenticação de utilizadores
- Gestão de restaurantes e menus
- Validação administrativa de restaurantes
- Pesquisa e filtragem de restaurantes

## Principais funcionalidades
- Sistema de autenticação (login, registo, sessões)
- Gestão de restaurantes e pratos
- Perfis de utilizador (cliente e administrador)
- Filtros de pesquisa e categorias
- Persistência de dados

## Tecnologias
- Node.js
- Express
- EJS
- MongoDB
- JavaScript
- HTML / CSS

## Arquitetura
- Padrão MVC (Model-View-Controller)

## Contexto académico
Projeto desenvolvido no ano letivo 2024/2025.
